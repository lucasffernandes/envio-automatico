# envio-automatico

Esse código faz o envio automátizado de mensagens para os
contatos na lista definida no código.

PASSO A PASSO PARA UTILIZAR ESSE CÓDIGO:

1° O wpp web deve estar logado no navegador padrão do dispositivo

2° ter instalado o 'Python'
  -comanados cmd:
    winget install Python.Python.3
    python --version

3° ter atualizado o 'pip' (para evitar erros)
  -comanados cmd:
    pip --version
    python -m pip install --upgrade pip

4° ter instalado a biblioteca 'pywhatkit'
 -comanados cmd:
   pip install pywhatkit
 -se der erro de instalação
   python -m pip install pywhatkit

PARA EXECUTAR O CÓDIGO:

1° no vs code: -> run (canto superior direito)

2° no cmd digitanto o caminho do código, e o nome do executável
   precedido da palavra: 'python'.
    Ex de caminho:

   "C:\Users\sg03530\Documents\Estudos\python\Envio automatizado wpp\python enviar_wpp.py"
